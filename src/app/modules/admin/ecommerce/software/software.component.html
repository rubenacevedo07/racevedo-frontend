Website Overview

Ecommerce

This is an e-commerce platform specializing in the sale of Star Wars ships.

  • 1- Starships: add your ships to the shopping cart
  • 2- Cart: check your list, make modifications and make the order
  • 3- Orders: check all your purchases

You can explore and purchase a variety of Star Wars ships while experiencing the different functionalities of the website, like user authentication, database integrity, API requests, and responsive design.

The website utilizes a modern and well-integrated technology platform:

  • Azure Service App for hosting the Java Spring Boot 3.0 backend in Docker containers.
  • Azure PostgreSQL for relational database management.
  • Strato Hosting for the Angular 17 frontend with Angular Material.
  • Security measures such as HTTPONLY and Secure cookies for protecting session data.
  • Azure Functions for serverless processing of API requests.

Please visit the site, test its features, and enjoy a seamless shopping experience.
